Privacy Policy

LongShot: Full Page Screenshot is a privacy-focused Chrome extension designed to capture full-page screenshots of webpages.

Data Collection

LongShot does not collect, store, transmit, or share any personal or sensitive user data.

Data Usage

  • All screenshot processing happens locally in the user’s browser.
  • No data is sent to external servers.
  • No analytics, tracking, or advertising services are used.
  • No personally identifiable information is collected.

Permissions

The extension uses the following permissions solely to provide its core functionality:

  • activeTab – Capture content from the currently active webpage after user interaction.
  • scripting – Inject capture-related scripts to scroll pages and handle fixed elements.
  • storage – Store temporary capture data and user preferences locally.
  • downloads – Allow users to save screenshots or exported PDF files.

Third-Party Access

No user data is sold, shared, or transferred to third parties.
